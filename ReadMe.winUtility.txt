사용법

1. winUtility.zip의 압축을 해제하여 한 폴더에 넣어놓습니다
2. winUtility.exe를 실행해봅니다
3. `(1 왼쪽에 있는 버튼)을 눌러봅니다
4. 마우스 옆에 나오는 툴팁을 봅니다
5. 툴팁이 있는 상태에서 1을 누를 경우 '현재 마우스가 있는 곳을 클릭'합니다
6. 다시 `를 눌러 툴팁이 있는 상태에서 2를 누를 경우 '엔터를 세번 입력'합니다
7. esc를 누르면 툴팁이 사라지고 f4를 누르면 winUtility 프로그램이 종료됩니다
8. ctrl + `를 누르면 winUtility.ini 파일이 열립니다


winUtility.ini

[common]
command = 

이 형식은 유지한 채로 command = 뒤에다가 만들고 싶은 매크로를 설정합니다
1, 2, 3,,, 9까지 입력 가능하며 각 매크로는 |(키보드의 백스페이스 옆에 있습니다) 키로 구분합니다

일반적인 글자(알파벳, 한글, 숫자)는 그냥 입력하면 되지만 {esc}, {enter}, {tab}, {f1}, {Numpad3}(키보드 오른쪽 숫자패드) 등은 {} 안에 입력해야 합니다
{} 안에 입력 가능한 문자는 https://autohotkey.com/docs/commands/Send.htm 페이지의 'Key Names' 항목에 나와있습니다.
주로 사용하는 문자는
{click} - 현재 마우스 위치를 한번 클릭
{click 3} - 현재 마우스 위치를 세번 클릭
{click, 100, 150} - 현재 활성화된 창의 100, 150 좌표를 한번 클릭
{enter}
{tab}
{home} {end} {pgup} {pgdn}
{up} {left} {down} {right}
!{left} - 알트 + 왼쪽 화살표. 크롬에서 뒤로가기 할때 사용합니다
{sleep 1} - 1/60초(약 16milliseconds)동안 대기합니다

반복 작업(녹스에서 열강 키고 상점 구매)이나 무한 클릭 등을 쉽게 만들어 쓰다가 혹시나 도움이 될까 싶어 공유 드립니다
스크립트 파일(winUtility.ahk)도 첨부하니 더 개선해서 사용해주세요-
